You must declare the attributes variable in the 'startup.txt' file The variables below were 
used in the sample, you can copy them to 'startup.txt' if you want tp try it out.

*create intelligence 0
*create agility 0
*create perception 0
*create luck 0
*create dexterity 0
*create charisma 0
*create stealth 0
*create strength 0

*To start the attributes distribuition screen enter the command 
"*gosub_scene characterAttributes attributes_distribution" anywhere you want.

Make sure 'characterAttributes.txt' is inside your project's 'scenes' folder in order for it to work.
